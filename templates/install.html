Installation

Linux

  1. Clone the repository: 
    git clone https://github.com/TegranGrigorian/nbody-gui
  2. Install nbody module from its repository: 
    git clone https://github.com/TegranGrigorian/n-body-simulator

    Your directory should look like this:

    .
├── nbody-gui
├───── * Some files
└── n-body-simulator
    ├── pyproject.toml
    ├── README.md
    ├── requirements.txt
    └── setup.py
  3. Navigate to the GUI project directory: 
    cd nbody-gui
  4. Create virtual environment (optional but recommended): 
    virtualenv .venv --python=python3
 source .venv/bin/activate
  5. Install required dependencies: 
    pip install -r requirements.txt

    Install the nbody module using pip command:

    pip install -e ../n-body-simulator

    NOTE: This is assuming that your directory looks like the one shown above. If it's different, change the file path ../n-body-simulator accordingly.

  6. Run the application: 
    python main.py

Windows

  1. Clone the repositories (same commands): 
    git clone https://github.com/TegranGrigorian/nbody-gui
 git clone https://github.com/TegranGrigorian/n-body-simulator
  2. Open PowerShell / CMD and navigate to the GUI folder: 
    cd nbody-gui
  3. Create a virtual environment (Windows): 
    python -m venv .venv
.venv\Scripts\activate
  4. Install required dependencies: 
    pip install -r requirements.txt

    Install the nbody module using pip (adjust the path if needed):

    pip install -e ..\n-body-simulator
  5. Run the application: 
    python main.py
