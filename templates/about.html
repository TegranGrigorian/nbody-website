About this website

A small site to demonstrate, document, and share the N-Body simulation work I built. Eventually, I will expand this site to contain downloadable packages, forums, version control, patch notes and more

What this site is

This website collects documentation, demos, and links for an N‑Body simulation project I created. It brings together a high-performance C++ physics engine, a Python GUI for composing and running simulations, and this website for documentation and demonstration.

Why I built the N-Body Simulator

I first read about chaotic gravitational systems in the context of the three-body problem and was fascinated by how small differences in initial conditions produce wildly different outcomes. I wanted to visualize that chaotic behavior interactively, so I built the engine and a GUI to compose scenes, run simulations, and capture the results.
