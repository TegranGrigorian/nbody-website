About this project

This project pairs a high-performance C++ N-body engine with a Python GUI that lets you compose, run, and capture gravitational simulations. The GUI is designed for rapid scene composition, reproducible presets, and easy capture of screenshots or short screencasts for teaching and demos.

  • Use cases: teaching, demonstrations, small research experiments
  • Works with presets or fully custom body setups

N-Body GUI — interactive editing and capture

Open the GUI to place bodies, tune parameters, and run the integrator in real time. The interface makes it easy to compose scenes and produce reproducible simulations.

Interactive GUI

Drag and drop bodies, configure properties, set auto-orbits, and capture the view for screenshots and demos.

High-performance Engine

Written in C++ for speed; bindings allow running large simulations with efficient integration schemes.

Reproducible Presets

Built-in presets (Solar System, test setups) help you create consistent scenes for teaching and demos.

Simulate Complex Systems

Recreate systems from literature or construct your own. For example, explore chaotic three-body interactions like those described in Cixin Liu's work — then capture the result as a short video.
