N-Body GUI — Quick Guide

How to use the graphical interface to build, run, and capture simulations.

Overview

The N-Body GUI provides an interactive environment to place bodies, configure simulation parameters, and run the integrator in real time. Use the canvas to drag bodies, the control panel to change parameters, and the playback tools to step or run the simulation.

Getting Started

    Open the GUI by running this command in your terminal at the GUI repository:

    python main.py

    If working properly, the GUI should open and look like the screenshot below.

Controls

God Mode

  • Left-click: select / place body
  • Preset: Preset of pre-made n-body systems
  • Clear All: Remove all Bodies

Simulation Mode

  • Play: run simulation
  • Pause: pause simulation
  • Stop & Return: Stop simulation and return to God Mode
  • Time Warp: Exponentially increase time steps
  • Auto CoG: Move camera to Center of Gravity (CoG)
  • Capture All: Camera moves to capture all bodies

Walkthroughs

Step-by-step examples to set up common systems in the GUI. These walkthroughs are intended to get you a presentable simulation quickly for screenshots or demos.

Earth–Sun System

  1. Left click on the black grid to Add Body
  2. Go into the Body Editor(left side) and name the body Sun, then in the mass section click Sun to set the mass of this body to a single Sun mass.
  3. Next, left click some where in the black grid, a new body should appear. In the body editor, call this body Earth and in the mass section click Earth to set it to Earth's mass.
  4. Ensure the body editor is still on Earth. Next click Auto Orbit then Select Target. Next click on the Sun to set it as the target for Earth's orbit. Lastly, click Apply Orbit and you should see a blue line originating from the Earth.
  5. Finally, press Play and you should be seeing Earth orbiting the Sun. If you want, you can use the view controls to adjust the camera. Your orbit should be similar to the one shown in the video below.

Preset Solar System

  1. Open the Presets menu and choose Solar System. This will populate the canvas with Sun, planets, and (optionally) major moons.
  2. Inspect the loaded bodies in the inspector panel. For cleaner screenshots, hide or remove small/distant bodies (asteroids) and focus on major planets.
  3. Increase canvas resolution and hide side panels (if supported) to produce a clean image.
  4. Run the simulation and use Time Warp or longer timesteps to show orbital motion quickly. Use the export feature to capture high-quality images.

Troubleshooting

  • If the GUI is unresponsive, try refreshing the page and restarting the simulation.
  • Large body counts can slow rendering — reduce particle count or increase timestep.
  • Check the browser console for errors and report them in the project's issue tracker.
