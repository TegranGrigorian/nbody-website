N-Body Simulation Documentation

Comprehensive guide and reference for the N-Body Simulation engine.

Introduction

The N-Body Simulation engine is a high-performance physics engine designed to simulate gravitational interactions between multiple bodies in space. This documentation provides an overview of the engine's features, API reference, and usage examples.

Getting Started

To get started with the N-Body Simulation engine, you can install it via pip:

pip install nbody-simulation

Once installed, you can create a simple simulation as follows:

from nbody_simulation import Simulation, Body
