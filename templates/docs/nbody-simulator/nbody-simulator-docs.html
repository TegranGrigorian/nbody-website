N-Body Simulation Documentation

Comprehensive guide and reference for the N-Body Simulation engine.

Introduction

The N-Body Simulation engine is a high-performance physics engine designed to simulate gravitational interactions between multiple bodies in space. This documentation provides an overview of the engine's design, library usage and more.

Structure

The engine is composed of 2 main classes:

  • Body: The body class manages the celestial body's properties and interactions.
  • Kosmos: The Kosmos class manages the universe in which the bodies exist. The universe manages bodies interactions and calculations.

The Kosmos class inherits a vector of "Body" objects. The Kosmos class is responsible for managing the simulation loop and updating all bodies within the universe. Along with the bodies, the "Kosmos" class handles the time steps as well. A figure of the inheritance structure is below.
