nbody-simulation

Core physics engine that computes gravitational interactions between bodies. Written for performance and designed to be used as a library by GUIs and analysis scripts. Provides APIs for stepping the simulation, adding/removing bodies, and exporting state snapshots.

nbody-gui

Graphical user interface for running and visualizing simulations produced by the nbody-simulation engine. Includes controls for initial conditions, real-time rendering, and recording/exporting simulations. Integrates with the engine via the provided Python bindings.
